# Stack Estándar - Skill Personalizada

Esta skill documenta el stack tecnológico estándar para todos los proyectos futuros.

## Uso

Para que el agente use esta skill automáticamente, simplemente menciona:

```
@stack-estandar-sitoct
```

O el agente la leerá automáticamente cuando:
- Inicies un nuevo proyecto
- Preguntes sobre tecnologías a usar
- Necesites asesoramiento sobre deployment

## Contenido

La skill documenta:

- ✅ Stack frontend (Vercel + Next.js)
- ✅ Stack backend (Cloud Run + .NET/Node.js)
- ✅ Base de datos (Supabase)
- ✅ CI/CD (GitHub Actions)
- ✅ Comparativa VPS vs Cloud
- ✅ Arquitecturas de referencia
- ✅ Workflow de deployment
- ✅ Checklist para nuevos proyectos

## Filosofía

**Usar siempre el mismo stack estándar** para garantizar:
- Consistencia entre proyectos
- Facilidad de mantenimiento
- Mejores prácticas del mercado
- Tier gratuito en la mayoría de servicios
